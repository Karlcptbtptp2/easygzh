# 主号视觉调性

这些规范会被翻译成主题 CSS（写入 current-theme.md 或仓库 themes/）。每次排版保持一致 = 调性稳定。

## 配色

- **主色**：#1a73e8（用于 H1/H2、强调 `<strong>`、链接）
- **正文色**：#333
- **引用/辅助色**：#666
- **背景**：纯白正文，引用块用 #f6f8fa 浅灰

## 字体与字号

- **正文字号**：15px
- **行高**：1.75（公众号长文舒适区）
- **字间距**：0.05em
- **字体栈**：苹方 / PingFang SC 优先，兜底系统无衬线

## 标题层级

- **H1**：22px，居中，底部一条 2px 主色实线
- **H2**：18px，左侧 4px 主色竖条
- **H3**：16px，纯主色

## 间距

- 段间距：15px
- 标题上下间距：上 1.4em / 下 0.6em
- 引用块上下：18px

## 分隔符

- `---` 渲染为居中虚线，宽度 50%

## emoji 风格

- 极少使用，仅在小标题处可加一个，正文不用
- 不用 ❤️🔥🎉 这类高饱和 emoji
