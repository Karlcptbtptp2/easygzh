## 正文

姐妹们谁懂啊😭
每次写完公众号文章，排版才是最痛苦的开始……

风格每次都不一样，手动调间距调配色调到怀疑人生
用排版工具吧，还要充会员、绑付费 API
排完还得手动复制粘贴到后台
主题就那几个，选来选去都撞衫
外链在正文里还点不了……

直到我发现了一个开源神器 —— easyGZH ✨

它是个 Agent Skill，装在你的 Agent 里（ZCode / Claude Code 这些都行）
你只需要把文章发给 Agent，说"帮我排版"
它就自动帮你排好，还能直接推到公众号草稿箱！

🤯 最让我震惊的是：它居然会**记住你号的风格**

排过一次之后，配色、字号、间距这些全都记下来了
以后每篇文章自动保持一致，再也不用每次重新调
这个能力其他排版工具真的都没有！！

👇 整个流程就三步：

1️⃣ 把文章发给 Agent（粘贴文字 / 给文件都行）
2️⃣ Agent 自动帮你排好版（套用你号的风格）
3️⃣ 推到草稿箱，后台直接能看到

就这么简单，跟 Agent 说话就行，不用学任何操作 💬

✨ 三个让我心动的点：

🧠 **记住风格** —— 排一次，以后都一致。这个是独家的，其他工具真没有
⚡ **本地渲染** —— 在你电脑上跑，不调任何付费 API，发布只用你自己的公众号凭证
💚 **开源免费** —— MIT 协议，代码全公开，想改就改，用着放心

对比一下那些排版工具：
❌ 每次风格漂移 → ✅ 排一次记住 forever
❌ 充会员绑付费 → ✅ 开源零成本
❌ 手动粘贴后台 → ✅ 一键推草稿箱
❌ 主题撞衫 → ✅ 真 CSS 随便改
❌ 外链点不了 → ✅ 自动转脚注

用了一段时间真心觉得：排版这件事，本来就不该这么累
让 Agent 帮你搞定，你只管写内容就好 📝
